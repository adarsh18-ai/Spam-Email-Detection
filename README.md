# Spam-Email-Detection
This project is a Machine Learning-based Spam Email Detection System developed using Python, Scikit-Learn, and Streamlit. The system analyzes the content of an email and predicts whether it is Spam or Not Spam.

The project uses a Bag-of-Words (BoW) feature extraction technique and a Multinomial Naive Bayes Classifier for classification.

Features
1) Detects Spam and Non-Spam Emails
2) User-friendly Streamlit Web Interface
3) Real-time Prediction
4) Displays Spam Probability Score
5) Lightweight and Fast Model

Machine Learning Pipeline

Email Text
→ Text Preprocessing
→ Bag-of-Words Feature Extraction
→ Multinomial Naive Bayes Classifier
→ Spam / Not Spam Prediction

Usage
Enter email text into the input box.
Click Check Spam.
View the prediction:
Spam Email
Not Spam Email
Check the Spam Probability score.

Sample Output

Input:

Congratulations! You have won a free iPhone. Click here to claim your prize.

Output:

SPAM EMAIL

Spam Probability: 0.97
